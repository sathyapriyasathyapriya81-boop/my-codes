# Daily Practice Code

A running log of my daily coding practice — Java fundamentals, DSA problems, and SQL queries — as part of my placement preparation.

## Why This Repo Exists

I'm a final-year B.Sc. Computer Science student preparing for campus placements. This repository is where I practice consistently instead of cramming last-minute — solving problems daily, tracking progress, and building the habit of showing up on GitHub every day.

## Topics Covered

- **Java:** Core syntax, OOP pillars (encapsulation, inheritance, polymorphism, abstraction), collections, exception handling
- **DSA:** Arrays, strings, searching & sorting, recursion, basic problem-solving patterns
- **SQL:** Queries, joins, normalization concepts (1NF–3NF)
- **HackerRank:** Problem-wise solutions with notes on approach

## How I Practice

Each solution folder typically includes:
- The problem statement (or a link to it)
- My solution code
- A short note on the approach or what I learned/got stuck on

I prioritize understanding *why* a solution works over just getting it to pass — the goal is to be able to explain my reasoning in an interview, not just produce working code.
